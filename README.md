
Name
====

**mcc.py** -  mod_cluster python client, allows registering non-java backends to mod_cluster balancers. 

Usage
=====

usage: mcc.py [-h] [-s SERVER] [-k KEY]
mcc.py: error: Must inform server address or advertise secret key
